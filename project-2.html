YelpCamp

YelpCamp is a RESTful web-app that allows users to view, post, comment on campgrounds worldwide.

Live Link

Project Overview

YelpCamp is a website where users can create and review campgrounds. In order to review or create a campground, you must have an account. This project was part of Colt Steele's web dev course on udemy.

    Features

  • Full CRUD for Campgrounds with Image Upload
  • Create & Delete Reviews for Campgrounds
  • Rate Campgrounds
  • User Authentication
  • Beautiful Cluster Map
  • Manage your Campgrounds
